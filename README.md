# Geek Food

Es una aplicación iOS de exploración gastronómica construida con SwiftUI. Conecta con una API en tiempo real para mostrar un catálogo de platos con información nutricional detallada, permite gestionar pedidos antes de confirmarlos y ofrece una pestaña de cuenta con perfil de usuario personalizable para una experiencia más precisa.

## Características técnicas 
**Arquitectura MVVM**<br/>
Separación clara entre lógica de negocio y vista para un código modular, mantenible y preparado para escalar.

**SwiftUI**<br/>
Interfaz construida con sintaxis declarativa — actualizaciones en tiempo real y compatibilidad multiplataforma nativa.

**Conectividad API**<br/>
Integración con API REST y decodificación JSON para mostrar contenido dinámico y actualizado en tiempo real.

**Optimización de recursos**<br/>
Carga asíncrona de imágenes con caché para un rendimiento óptimo y una experiencia de usuario fluida.

**Gestor de persistencia**<br/>
Almacenamiento y recuperación de preferencias y configuración del usuario mediante gestión optimizada de datos.

**Gestión de errores**<br/>
Tratamiento explícito de errores para mantener una experiencia fluida y sin interrupciones.

**Alertas personalizadas**<br/>
Notificaciones contextuales que mantienen al usuario informado en cada paso de la interacción.

**Vistas de estado vacío**<br/>
Contenido visual relevante cuando no hay datos disponibles, evitando pantallas en blanco y mejorando la UX.

**Diseño UI/UX**<br/>
Interfaz intuitiva y visualmente cuidada que facilita el uso natural de la aplicación.
